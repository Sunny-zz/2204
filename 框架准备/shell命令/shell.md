## shell 命令的基本使用
命令行是根据所在位置执行命令的

### 命令行工具
cmd   powershell   gitbash  

### 基础命令
输入命令之后 回车就是执行命令

- pwd  查看当前所在位置
- ls   查看当前目录下有哪些内容   
    - 参数  -a   可查看隐藏内容
- cd   跳转目录
- clear 清屏
- mkdir 创建文件夹
- touch 创建文件 文件名必须写后缀
- rm 删除文件
  - 参数 -r 删除文件夹
  - 参数 -f 强制删除   一般不会用到
- mv 移动内容    `mv  移动的内容  移动到哪`
- cp 拷贝文件   `cp  拷贝的内容  拷贝到哪`
  - 参数 -r 拷贝文件夹
- cat 查看文件的文本内容  


### 快捷键
- ctrl + c 结束当前
- 上下箭头 查看历史输入的命令
